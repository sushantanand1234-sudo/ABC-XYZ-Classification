# 📦 ABC–XYZ Inventory Classification & Pareto Analysis

> An end-to-end inventory analytics project that applies **ABC Analysis**, **XYZ Classification**, and the **Pareto Principle** to prioritize inventory, evaluate demand variability, and support data-driven inventory management decisions.

---

## 📖 Project Overview

Efficient inventory management requires identifying **which products generate the highest business value** and **which products have predictable demand**.

This project analyzes **1,000 unique SKUs** across **5 product categories** using:

- **ABC Analysis** – Identifies high-value inventory based on revenue contribution.
- **XYZ Classification** – Measures demand variability using the Coefficient of Variation (CV).
- **ABC–XYZ Matrix** – Combines value and demand variability for inventory prioritization.
- **Pareto Analysis** – Demonstrates the 80/20 principle to highlight the small set of SKUs responsible for most revenue.

The project concludes with business insights and inventory recommendations supported by clear visualizations.

---

---

# 🎯 Business Problem

Inventory managers must answer questions such as:

- Which products generate the highest revenue?
- Which products have stable or unpredictable demand?
- Which SKUs require the highest inventory priority?
- How can inventory costs be reduced without affecting customer service?

This project provides a structured framework to answer these questions using inventory analytics.
